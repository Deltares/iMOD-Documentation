****************
iMOD QGIS plugin
****************
The iMOD plugin extends QGIS functionalities to assist with groundwater modeling, 
by facilitating 4D geospatial data exploring and linking to the iMOD 3D viewer.

Primary components are:

* Connecting to the iMOD 3D viewer
* Timeseries visualization
* Cross-section visualization
* Reading .IPF files
* Easy viewing of NHI data
