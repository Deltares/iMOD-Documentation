**************
iMOD 3D Viewer
**************

The iMOD 3D viewer can visualize unstructured grids (UGRID)
as well as structured grids (``.idf``) in 3D. 
To fully utilize the functionality of the 3D viewer,
the iMOD QGIS plugin is required as well.
